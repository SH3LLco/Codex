# Virtual Environments

## Creating the Virtual Environment
### Windows Powershell
``` powershell
py -m venv venv\\
```

### Linux/MacOS
``` bash
python3 -m venv venv/
```

## Activating the Virtual Environment
### Windows Powershell
``` powershell
venv\\Scripts\\activate
```

### Linux/MacOS
``` bash
source venv/bin/activate
```

## Deactivating the Virtual Environment
### Windows Powershell
``` powershell
deactivate
```

### Linux/MacOS
``` bash
deactivate
```
